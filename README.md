## About the Project

The **T3 Chat Cloneathon** is the first-ever event of its kind, focusing on the creation of a clone for T3 Chat. This project aims to foster collaboration among developers and enhance coding skills. As you work on this project, you'll gain experience, showcase your talent, and have a chance to win prizes.

This repository is a work in progress, and more features and updates will be added soon. Stay tuned!

## Getting Started

To get started with the T3 Chat Cloneathon, follow these steps:

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Killercroc01/t3-cloneathon.git
   cd t3-cloneathon
   ```

2. **Install Dependencies**
   Make sure you have the necessary dependencies installed. You can do this by running:
   ```bash
   npm install
   ```

3. **Run the Project**
   Start the project with the following command:
   ```bash
   npm start
   ```

4. **Open in Browser**
   Visit `http://localhost:3000` to see your chat clone in action.

## Technologies Used

- **JavaScript**: For front-end and back-end development.
- **Node.js**: Server-side runtime.
- **Express**: Web framework for Node.js.
- **Socket.IO**: For real-time communication.
- **MongoDB**: Database for storing user data and messages.
